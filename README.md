# ECNmsUtils

This is a wrapper intended to be used inside https://github.com/EverNife/EverNifeCore to make my life easier!

It creates a easy way to compile NMS dependent code for different types of Spigot Platforms and Java Versions.

Right now supports:

- 1.7.10 (compiled in java 8)
- 1.12.2 (compiled in java 8)
- 1.16.5 (compiled in java 8)
- 1.20.1 (compiled in java 21)
- 1.20.2 (compiled in java 21)
- 1.21.1 (compiled in java 21)
